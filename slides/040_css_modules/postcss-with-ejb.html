Offline transformation

css-modules/postcss-modules を利用することで
CSS Modulesの事前変換が可能 

/* styles.css */
.myClass {
  color:red;
}


/* converted_styles.css */
._myClass_xxx_yyy {
  color:red;
}
// converted_styles.json
{
  "myClass": "_myClass_xxx_yyy"
}

出力されたjsonから対応関係を参照することで
他言語(e.g. Rails)からもCSS Modulesを利用できる
