##  How to create Component?

多くのJS FrameworkがUI Componentの開発をサポート

<ul class="good">
  <li>
   **React**:
<span class="smaller">
  Stateless Function , `React.Component`
</span>
</li>
  <li>AngularJS:
<span class="smaller">
  `angular.module(...).component(...)`
</span>
</li>
  <li>
Vue.js:
<span class="smaller">
  `Vue.component(...)`
</span>
</li>
<li class="no-mark">and more...</li>
</ul>
